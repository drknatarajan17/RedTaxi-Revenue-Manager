# 🚖 RedTaxi Revenue Manager

A professional Streamlit-based business management application designed for taxi owners to monitor daily revenue, expenses, profitability, and business performance.

---

## 📌 Project Overview

RedTaxi Revenue Manager helps taxi owners maintain daily business records, calculate profits automatically, analyze expenses, and generate reports for better financial decision-making.

The application is developed using Python, Streamlit, Pandas, and Matplotlib.
